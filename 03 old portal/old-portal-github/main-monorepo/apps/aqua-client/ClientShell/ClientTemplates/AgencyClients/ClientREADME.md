# AgencyClientsView

**Suite:** Revenue Suite  
**Section:** AQUA Operations → Revenue Hub

## What it does
High-level agency client roster. Displays all active client accounts with their current stage, assigned team, revenue contribution, and suite access. Provides quick-access actions for provisioning and managing client portals.

## Key features
- Client roster with stage and health indicators
- Revenue per client summary
- Quick portal access and impersonation
- Suite enablement controls
