# FulfillmentView

**Suite:** Operations Suite  
**Section:** AQUA Operations → Operations

## What it does
Client fulfilment and project delivery management. Tracks active projects, task lists, team assignments, and delivery milestones. Feeds directly from client records and the project pipeline.

## Key features
- Project board with status columns
- Task list per project with assignees
- Delivery milestone tracker
- Team workload view
