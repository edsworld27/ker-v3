# Media storage

Stray images that landed in the repo root, sorted by what they
belong to. Moved here for tidiness — they aren't served by the app
(those live under `/public`).

```
media-storage/
  felicia/   – Luv & Ker / Odo by Felicia brand assets
  aqua/      – Aqua portal screenshots, mockups, brand kits
  misc/      – uncategorised
```

If you need to reference an image from the running app, copy it to
`/public` first — that's the folder Next.js serves at the URL root.
